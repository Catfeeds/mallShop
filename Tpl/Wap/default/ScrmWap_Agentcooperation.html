什么是人来风合伙人

人来风合伙人是为帮助人来风入驻商户
创造更大价值的重要力量。

人来风是专注于为广大商户提供数据价值，帮助入驻商户挖掘移动红利的SCRM平台。为保障人来风商户获得更完善更有价值的服务，提升商户平台应用的有效性，强化人来风合伙人服务能力及盈利能力规范人来风SCRM市场定价，实现增值服务商品化，促进人来风SCRM平台市场生态体系正向发展，现对人来风合伙人原有身份体系进行规范，需满足以下条件，获得对应合作等级权益。

展业伙伴
等级获取条件
  1. 个人及企业资质均可
  2. 签订展业伙伴合作协议
  3. 获取展业邀请码
等级权益
  1. 分润权益
  2. 首次入驻：900元/年/商户
    首次入驻：1500/2年/商户

    续约续约：1500/2年/商户
    (每月5号结算，永续分润）
高级展业伙伴
等级获取条件
  1. 必须具备企业资质
  2. 签订展业伙伴合作协议
  3. 获取展业邀请码
  4. 累积展业成单3单且对应3家签约商户真实粉丝数大于500，会员数大于100，销售额大于1000元且线上收入不低于30%
等级权益
  1. 分润权益
  2. 首次入驻：1800元/年/商户
    首次入驻：3000元/2年/商户
  3. 续续约约：1600元/年/商户
    续约续约：3000元/2年/商户
    (每月5号结算，永续分润)
服务伙伴
等级获取条件
  1. 个人及企业资质均可
  2. 签订展业伙伴合作协议
  3. 当月累计展业成交5单或对应签约商户SCRM5统计销售总额达20000元且线上收入不低于30%
  4. 具备增值业务服务能力
等级权益
  1. 分润权益：单笔分润不低于30%
  2. 增值服务权益
  3. 地域保护
  4. 商户资源分配
  5. 品牌授权
  6. 区域市场推广支持

  1. 当月累计展业成交10单并且对应签约商户SCRM5统计销售总额达20000元且线上收入不低于30%
  2. 全职
  3. 具备增值业务服务能力
  4. 具备区域公司管理能力
等级权益

  4. 30万及以上年薪
