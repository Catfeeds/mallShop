测测你承受挫折能力有多强？

人生在世，不可能一帆风顺，相信每个人都品尝过挫折的滋味。每个人应对挫折的能力都是不一样的，通过以下测试，你可以了解自己对挫折的应对能力。请做出选择。
开始测试

1：面临问题时，你会：

  • 知难而进。
  • 找人帮助
  • 放弃目标

2：你对自己才华和能力的自信程度如何?

  • 知难而进。
  • 找人帮助
  • 放弃目标

3：每次遇到挫折，你：

  • 大部分都能自己解决。
  • 有一部分能解决
  • 大部分解决不了

4：在过去的一年中，你遭受几次挫折：

  • 0～2次
  • 3～5次
  • 5次以上

5：碰到难题时，你：

  • 失去自信
  • 为解决问题而动脑筋
  • 介于A、B之间

6：产生自卑感时，你：

  • 不想再工作。
  • 立即振奋精神工作
  • 介于A、B之间

7：困难落到自己头上时，你：

  • 厌恶至极。
  • 认为是个锻炼机会
  • 介于A、B之间

8：碰到讨厌的对手时，你：

  • 无法应付。
  • 应付自如
  • 介于A、B之间

9：工作中感到疲劳时：

  • 总是想着疲劳，脑子不好使了。
  • 休息一段时间，就忘了疲劳
  • 介于A、B之间

10：有令你担心的事时，你：

  • 无法工作。
  • 工作照样不误
  • 介于A、B之间。

11：工作进展不快时，你：

  • 焦躁万分。
  • 冷静地想办法
  • 介于A、B之间

12：面临失败，你：

  • 破罐子破摔 。
  • 将失败转化为成功
  • 介于A、B之间

13：工作条件恶劣时，你：

  • 无法做好工作 。
  • 能克服困难做好工作
  • 介于A、B之间

14：上级给了你很难完成的任务时，你会：

  • 顶回去了事 。
  • 千方百计做好
  • 介于A、B之间

测试结果：0-8分

详细分析:

说明你不能承受挫折的打击，遇到一点挫折就不知所措，灰心失望。 建议多参加一些锻炼意志和承受能力的活动，比如体育活动，各种比赛，读一些励志的书籍，学习在失败中不断提高自己抗挫折的能力。并且交一些意志坚强、性格乐观的朋友，他们会在你遇到挫折的时候可以给予适当的建议和鼓励。当然你还可以找心理医生咨询，针对你个人的具体情况提出相应的改进方案。

详细分析:

说明你对某些挫折打击有一定的承受能力，但是你在遇到某些挫折的时候仍然会表现出脆弱。 建议遇到挫折的时候，多往事情有利的方面想，能冷静分析情况后再做出决定，比如挫折产生的根源，自己能否解决，或者是否值得。
